С помощью киберслизняка учёные хотят детально изучить человеческий мозг. У смоделированного хищника есть зачатки самосознания, он способен связать мотивацию с воспоминаниями и реагировать на внешние ситуации. «Морская слизь — хорошая модель древней системы, которая ещё присутствует в наших мозгах» — заявляют исследователи.
